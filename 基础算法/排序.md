## 排序和常见问题

| 算法     | 时间复杂度 | 常见问题                 | 排序稳定性* |
| -------- | ---------- | ------------------------ | ----------- |
| 快速排序 | O(NlogN)   | 找第k大的数，找k个大的数 | 不稳定      |
| 归并排序 |            | 链表排序                 | 稳定        |
|          |            |                          |             |

*排序稳定性指的是，对于值相同的多个元素，排序后不改变相对顺序。
